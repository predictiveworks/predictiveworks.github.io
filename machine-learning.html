Works ML Plugins

The main objective of PredictiveWorks. machine learning support is to make Apache Spark ML available as plugin components for CDAP data pipelines.

This responds to the vision of Visual ML - Code-free and pluggable orchestration of ML data pipelines for a wide variety of business use cases.
